# Fractal
21 school project, Fractals drawing

Use "make" to compile the program
then "./fractol fractal_type_lowercased"
Use "1"/"2" keys to increase/decrease number of iterations

types of fractals:
1. Mandelbrot
2. Julia
3. Burning_ship
4. Spider
5. Mishaniabrot
